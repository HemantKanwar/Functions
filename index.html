Q6 => Print output
var x = 21; a(); b(); function a() { x = 20; console.log(x); }; function b() { x = 40; console.log(x); }; 
            Ans : The Output will be 20 ,40 . Both a() and b() modify the value of the global variable x, and then
             they each print the updated value using console.log(). As a result, you get 20 printed when a() is called, 
             and then 40 printed when b() is called.
Q7 => Write a function that accepts parameter n and returns factorial of n. 
            Ans : let number = 5;
            let result = factorial(number);
            
            function factorial(n) {
                if (n === 0 || n === 1) {
                  return 1;
                } else {
                  return n * factorial(n - 1);
                }
              }
            
            console.log(`Factorial of ${number} is ${result}`);
